
This directory contains sources to build documentation and web site of
Pyleaf.

- gensite.py is a python script building Pyleaf site or
  documentation. (Edit to switch doc/site).

- template.raw contains the site backbone (header, footer, etc.) for
  documentation.

- templateH.raw (NOT distributed since it includes private contents)
  contains the site backbone (header, footer, etc.) for Pyleaf site.

- other .raw files contain site/doc pages.

- the contents of the directory "copyme" should be added to site or
  doc root after building them.
